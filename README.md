# 📸 Django Instagram Clone
https://django-instagram-clone-v7fu.onrender.com

A full-stack social media web application inspired by Instagram, built using Django.

This project includes authentication, image & video uploads, AJAX-based interactions, and a modern responsive UI.

---

## 🚀 Live Features

- 🔐 User Authentication (Signup/Login/Logout)
- 👤 User Profile Page
- 📸 Image Post Upload
- 🎥 Video Post Upload
- ❤️ Like System (AJAX)
- 💬 Comment System
- ⚡ Dynamic Feed Updates
- 📱 Responsive UI Design

---

## 🛠️ Tech Stack

- Backend: Django (Python)
- Frontend: HTML, CSS, JavaScript
- AJAX for dynamic interactions
- Database: SQLite (Development)
- Media Handling: Django Media System
